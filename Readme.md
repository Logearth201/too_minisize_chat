日本では、今や2019年となるのに、いまだに3MBくらいしか無料レンタルサーバーは利用できません。
そんな要件で掲示板を作ったらおそらくこんな感じになります

※ルール：
・外部JSファイルなどの読み取り禁止
・ES6での記載禁止
・PHP4までしか利用できない
・500request/hを超える「高負荷」サービスにしないこと。
・5MB/hの通信料を超える「高負荷」サービスにしないこと。
